# Launch Checklist

## Pre-Launch (1 week before)
- [ ] Product is deployed and stable
- [ ] Analytics tracking verified (see `analytics/tracking-plan.md`)
- [ ] Sentry error monitoring active
- [ ] Landing page SEO optimized (title, meta, OG image)
- [ ] Screenshots captured → `marketing/assets/screenshots/`
- [ ] OG image created → `marketing/assets/og-image.png`
- [ ] Demo recording/GIF prepared → `marketing/assets/demo/`
- [ ] Product Hunt listing drafted → `marketing/launch/product-hunt.md`
- [ ] Show HN post drafted → `marketing/launch/show-hn.md`
- [ ] Reddit posts drafted → `marketing/launch/reddit.md`
- [ ] X launch thread drafted → `marketing/content/social/launch-thread.md`
- [ ] Email to waitlist prepared → `marketing/content/email/launch-announce.md`
- [ ] FAQ ready → `ops/cs/faq.md`

## Launch Day
- [ ] Product Hunt: publish at 12:01 AM PST
- [ ] Show HN: post during US morning hours
- [ ] X: publish launch thread
- [ ] Reddit: post in relevant subreddits (1-2 max, respect rules)
- [ ] Email: send launch announcement to waitlist
- [ ] LinkedIn: share launch post (if B2B)
- [ ] Respond to ALL comments/feedback within 2 hours

## Post-Launch (first week)
- [ ] Daily: check analytics dashboard (5 min)
- [ ] Daily: respond to all user feedback
- [ ] Day 2: publish "Day 1 results" build-in-public post
- [ ] Day 3: post follow-up content (lessons learned, user reactions)
- [ ] Day 7: write first weekly report → `analytics/reports/`
- [ ] Day 7: decide Kill / Keep / Scale (see `analytics/kill-criteria.md`)
